功能

modbus工具软件,软件分为4部分功能：
1. 实现modbus主机的06、10、03、04号指令，对设备进行调试
2. 支持周期任务自动发送
3. 可将读数显示成曲线
4. 通信数据通过彩色显示，区分不同的数据含义
软件由配置文件组织，配置文件描述周期任务和数据的类型、曲线名称

使用说明

单次发送

在单次发送界面中，填写发送地址、寄存器地址（10进制），然后选择发送类型，支持的发送类型包括：写单寄存器(0x06)、写多寄存器(0x10)、读保持寄存器(0x03)、读输入寄存器(0x04)。
